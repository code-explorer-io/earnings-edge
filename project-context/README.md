## Quick Reference

### Current Status
- **API Tier**: FREE (single quarter per ticker)
- **Deployment**: Vercel (auto-deploy from main)
- **Domain**: earningsedge.io

### Key Limitations (FREE Tier)
- Only latest earnings call available
- Some tickers unavailable (SNOW, COIN, etc.)
- No historical quarter comparison

See [API_INTEGRATION.md](API_INTEGRATION.md) for details.
